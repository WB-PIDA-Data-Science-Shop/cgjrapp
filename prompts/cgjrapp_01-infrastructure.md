# Sprint 1: Package Infrastructure

## Objective
Lay the structural foundation of `cgjrapp` so that every subsequent sprint builds on a clean, consistent base. Nothing visual is produced in this sprint — it is pure scaffolding. This sprint is complete when the package loads cleanly, passes `devtools::check()` with 0 errors/warnings, and the infrastructure tests pass.

## Context
- The app is an R package. Entry point is `run_cgjrapp()`.
- Data comes exclusively from `cgjrdata` (lazy-loaded: `institutional_averages_tbl`, `ctfdata_list`, `rawdata_list`, `metadata_tbl`).
- UI is built with `bslib` + `shiny`. Plots will be `ggplot2` wrapped with `plotly::ggplotly()`.
